# Not@MRP Backend

## Setup

1. Copy `.example.env` to `.env` and adjust values as needed:
   
   ```bash
   cp .example.env .env
   ```
3. Install dependencies:
   ```bash
   npm i
   ```
4. Build the project:
   ```bash
   npm run build
   ```
5. Start the server:
   ```bash
   npm start
   ```

The API documentation is available at `http://localhost:3000/docs` once the server is running.
