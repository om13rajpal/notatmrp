# Not@MRP Backend

## Setup

1. Copy `.example.env` to `.env` and adjust values as needed:
   
   ```bash
   cp .example.env .env
   ```
3. Install dependencies:
   ```bash
   npm i
   ```
4. Build the project:
   ```bash
   npm run build
   ```
5. Start the server:
   ```bash
   npm start
   ```
